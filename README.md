# Лендинг для диагностики заявок застройщиков

В репозитории есть готовый к публикации лендинг:

- `index.html` — основной файл для GitHub Pages.
- `prototypes/home-builders-lead-gen-landing.html` — архивная версия первого HTML-прототипа.

## Как открыть локально

Вариант 1: откройте `index.html` двойным кликом в браузере.

Вариант 2: запустите локальный сервер из корня репозитория:

```bash
python3 -m http.server 4173
```

Затем откройте:

```text
http://127.0.0.1:4173/
```

## Как опубликовать через GitHub Pages

1. Закоммитьте `index.html` в основную ветку репозитория.
2. Откройте настройки репозитория на GitHub.
3. Перейдите в раздел Pages.
4. В качестве источника выберите публикацию из ветки и корневой папки репозитория.
5. Сохраните настройки и дождитесь публикации.

После публикации GitHub Pages будет открывать `index.html` как главную страницу.

## Что заменить перед публикацией

В `index.html` замените placeholder-ссылку Telegram:

```text
https://t.me/your_username
```

Также замените временный адрес в Open Graph метатеге `og:url`, если будете использовать превью страницы в соцсетях и мессенджерах.
